# Unicode Visualizer Plugin

A plugin to debug/visualize what's going on in a Unicode string.

## Updating src/ucd/

1. Have lua 5.1
2. Download the latest version of UCD http://www.unicode.org/Public/UCD/latest/ucd/UCD.zip
3. Unzip it into `UCD/`
4. Run `lua build.lua`
